## 感謝你的參與

### 這項研究其實在問什麼
我們在比較四種「決策輔助方式」對你**判斷品質**的影響：
1. 完整工具（統計概率 + 結構化反思 + 符號介面）
2. 同上但沒有符號介面
3. 一段**刻意通用、不含具體建議**的「解讀」
4. 不給任何輔助

你被**隨機**分到其中一組。分組是隨機的，與你個人特質無關。

### 關於「解讀」的重要說明 ⚠️（給第 3 組與所有人）
如果你看到的是一段感覺「蠻準」的個人化解讀——請知道：
**那是刻意設計的通用語句，對幾乎任何人都會「感覺準」。**
這叫**巴納姆／福勒效應**（Forer, 1949）：模糊、正向、適用於多數人的描述，
會讓人誤以為是針對自己。我們用它來檢驗：真正有用的決策工具，
必須做到比「讓人感覺準」更多的事（也就是真正改善判斷的校準度）。

### 本工具不預測命運
即使是完整工具，也**不預測**你個人的結果——研究已知個體高後果結果存在
難以突破的預測天花板。它的目的是幫你**想得更全、管理不確定性**，而非告訴你答案。
